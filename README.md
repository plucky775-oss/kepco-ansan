# 전기공급약관 챗봇

이 파일은 업로드된 `기본공급약관.pdf`를 바탕으로 만든 **오프라인형 검색 챗봇**입니다.

## 사용 방법
1. 압축을 풉니다.
2. `index.html` 파일을 엽니다.
3. 질문을 입력하거나 빠른 질문 버튼을 눌러 사용합니다.

## 특징
- 기본공급약관 본문 조문 + 별표 일부를 지식베이스로 내장
- 명의변경, 해지, 요금, 이의신청, 시설부담금 등은 우선 규칙형 답변 제공
- 그 외 질문은 관련 조문을 검색해 요약과 원문 발췌를 보여줌
- 별도 서버 없이 브라우저에서 바로 실행 가능

## 주의
- 이 버전은 시제품입니다.
- 최종 민원처리, 실제 계약, 요금 확정 판단은 최신 공식 약관과 한전 안내를 다시 확인하세요.
